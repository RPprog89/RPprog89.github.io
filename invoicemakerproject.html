Invoice Maker



Esta es una aplicación enfocada a la gestión de recursos de una empresa, como un sencillo ERP, permitiendo almacenar y gestionar datos de clientes y productos, ademas de generar y almacenar facturas.

Cuenta con un dashboard que informa de algunos datos generales en forma de gráficas y tarjetas, y diferentes secciones donde gestionar cada uno de los recursos. Permite ademas generar facturas en PDF.

Es un programa de escritorio desarrollado en Java con una base de datos mysql en local que debe ser instalada junto con el programa.
