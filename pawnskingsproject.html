Pawns & Kings



Este proyecto trata de una aplicación móvil para el sistema operativo Android que permite a los usuarios jugar al ajedrez con otros usuarios en tiempo real en diferentes modalidades de tiempo de juego, así como guardar las partidas jugadas entre estos para poder rescatarlas y revisarlas desde la aplicación móvil posteriormente. También proporciona un modo de juego offline contra una IA construida desde cero y aplica un sistema de puntuación para los usuarios que haga más interesante el juego.

Consta de una parte cliente, la aplicación móvil, escrita en Java con las librerías de Android Studio, y otra parte servidora tambien escrita en Java con la ayuda del framework SpringBoot para la implementación de la Api REST, con una base de datos en mysql. El servidor además de gestionar las peticiones HTTP para el registro y login de usuarios, el guardado y recuperación de partidas...etc también hace de servidor socket para las partidas entre usuarios, emparejandolos y reenviando la información entre un cliente y otro por medio de conexiones TCP durante una partida.

La IA o motor de ajedrez a sido implementada en el cliente desde cero utilizando algoritmos de búsqueda en el árbol de juego tales como Minimax y poda alfa-beta para la posterior evaluación de las posiciones.
